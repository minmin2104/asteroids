## Build

**REQUIREMENTS**
- Python 3.12

```
git clone https://github.com/minmin2104/asteroids.git
cd asteroids
python -m venv .venv
```

Then activate the environment

- On Windows

```
.\.venv\Scripts\activate
```

- On Linux

```
source ./.venv/bin/activate
```

Install the requirements

```
pip install -r requirements.txt
# Run the game
python main.py
```
